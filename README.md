### 1.2参数支持

|  参数名  | 参数描述                                      | 参数类型 | 默认值 |
| :------: | --------------------------------------------- | -------- | :-----: |
|   type   | 按钮类型(primary/success/warning/danger/info) | string   | default |
|  plain  | 是否为朴素按钮（镂空）                        | boolean  |  false  |
|  round  | 是否为圆角按钮                                | boolean  |  false  |
|  circle  | 是否为圆形按钮                                | boolean  |  false  |
| disabled | 是否禁用按钮                                  | boolean  |  false  |
|   icon   | 图标类名                                      | string   |   无   |
